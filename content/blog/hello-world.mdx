## 博客上线了！

经过一番折腾，我的个人博客终于搭建好了。

### 为什么选择 Next.js？

- **React 生态** — 丰富的组件和动画库支持
- **静态生成** — 构建后纯静态，加载极快
- **MDX 写作** — 支持在 Markdown 中嵌入 React 组件
- **炫酷交互** — GSAP + Framer Motion 加持

### 技术栈

| 组件 | 选择 |
|---|---|
| 框架 | Next.js 16 |
| 样式 | Tailwind CSS v4 |
| 动画 | Framer Motion + GSAP |
| 部署 | Vercel |
