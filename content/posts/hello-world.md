## 博客上线了！

经过一番折腾，我的个人博客终于搭建好了。

### 技术栈

| 组件 | 选择 | 费用 |
|---|---|---|
| 静态生成器 | Hugo | 免费 |
| 主题 | PaperMod | 免费 |
| 托管 | GitHub Pages | 免费 |
| SSL 证书 | GitHub Pages 自带 | 免费 |

### 为什么选择 Hugo？

- **构建速度快** — 几百篇文章也能秒级构建
- **无需数据库** — 纯静态，安全稳定
- **Markdown 写作** — 专注内容本身
- **主题丰富** — PaperMod 简洁优雅
